# Welcome to Zen LM

Zen LM is a family of next-generation language models built on Qwen3 architecture, focusing on delivering high-performance, efficient, and open-source AI across multiple scales.

## Why Zen LM?

### 🚀 Ultra-Efficient
From 0.6B to 480B parameters. Optimized for deployment on phones, laptops, and AI supercomputers.

### 🔒 Truly Private
100% local processing. No accounts, no telemetry, no tracking. Your data stays on your device.

### 🌱 Environmentally Responsible
95% less energy than cloud AI. Carbon-negative operations supporting environmental preservation.

### 💚 Free Forever
Apache 2.0 licensed. No premium tiers, no API fees, no hidden costs.

## Quick Start

```python
from transformers import AutoModelForCausalLM, AutoTokenizer

model = AutoModelForCausalLM.from_pretrained("zenlm/zen-nano")
tokenizer = AutoTokenizer.from_pretrained("zenlm/zen-nano")

inputs = tokenizer("Hello, I am", return_tensors="pt")
outputs = model.generate(**inputs, max_length=100)
print(tokenizer.decode(outputs[0]))
```
