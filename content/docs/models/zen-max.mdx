# Zen Max

Zen Max is the largest model in the Zen family — a 1T+ reasoning-first language model built for test-time scaling through extended thinking and tool-calling. With 32B active parameters via MoE sparse activation and up to 128K thinking tokens per step, it executes 200-300 sequential tool calls without human intervention for agentic reasoning, search, coding, and mathematical problem solving.

## Specifications

| Property | Value |
|----------|-------|
| Total Parameters | 1.04T (1,044B) |
| Active Parameters | 32B (MoE, 384 experts, top-8 routing) |
| Architecture | Zen MoDE (MoE) |
| Context Window | 256K tokens |
| Thinking Capacity | 96K-128K tokens per step |
| License | Apache 2.0 |
| Formats | SafeTensors (BF16), INT4 (native QAT) |

## Key Capabilities

- **Agentic reasoning** — Extended chain-of-thought with `<think>` tags, up to 128K thinking tokens
- **Agentic search** — 200-300 sequential tool calls, real-world information gathering
- **Agentic coding** — Multi-language support, terminal automation, component-heavy web development
- **Mathematical reasoning** — PhD-level math with Python tool integration
- **Heavy Mode** — 8 parallel reasoning trajectories with reflective aggregation

## Quick Start

```python
from transformers import AutoModelForCausalLM, AutoTokenizer

model = AutoModelForCausalLM.from_pretrained("zenlm/zen-max")
tokenizer = AutoTokenizer.from_pretrained("zenlm/zen-max")

messages = [
    {
        "role": "user",
        "content": "Research and analyze the latest developments in quantum computing, then write a comprehensive report."
    }
]

# Model will think, search, browse, and synthesize autonomously
response = model.chat(tokenizer, messages, thinking_budget=128000, max_tool_calls=300)
```
