# Zen Coder

Zen Coder is Zen LM's family of code-focused AI models spanning three capability tiers. The flagship `zen-coder-480b-instruct` is a 480B-parameter Mixture of Experts model with 35B active parameters, delivering state-of-the-art results on agentic coding, browser-use, and tool-use benchmarks. The family supports 358 programming languages, fill-in-the-middle (FIM), function calling, and extended chain-of-thought reasoning.

## Model Family

| Model | Parameters | Active | Context | Use Case |

## Quick Start

### Flagship (480B)

```python
from transformers import AutoModelForCausalLM, AutoTokenizer

model = AutoModelForCausalLM.from_pretrained(
    "zenlm/zen-coder-480b-instruct",
    torch_dtype="auto",
    device_map="auto"
)
tokenizer = AutoTokenizer.from_pretrained("zenlm/zen-coder-480b-instruct")

messages = [{"role": "user", "content": "Write a quicksort algorithm in Python with type hints."}]
text = tokenizer.apply_chat_template(messages, tokenize=False, add_generation_prompt=True)
inputs = tokenizer([text], return_tensors="pt").to(model.device)

generated_ids = model.generate(**inputs, max_new_tokens=4096)
output = tokenizer.batch_decode(
    [generated_ids[0][len(inputs.input_ids[0]):]],
    skip_special_tokens=True
)[0]
print(output)
```

### Edge (4B)

```python
from transformers import AutoModelForCausalLM, AutoTokenizer

model = AutoModelForCausalLM.from_pretrained("zenlm/zen-coder", device_map="auto")
tokenizer = AutoTokenizer.from_pretrained("zenlm/zen-coder")

messages = [{"role": "user", "content": "Write a binary search in Rust."}]
text = tokenizer.apply_chat_template(messages, tokenize=False, add_generation_prompt=True)
inputs = tokenizer([text], return_tensors="pt").to(model.device)
output = model.generate(**inputs, max_new_tokens=1024)
print(tokenizer.decode(output[0][len(inputs.input_ids[0]):], skip_special_tokens=True))
```

## Fill-in-the-Middle (FIM)

```python
input_text = """<|fim_prefix|>def quicksort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    <|fim_suffix|>
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quicksort(left) + middle + quicksort(right)<|fim_middle|>"""

messages = [
    {"role": "system", "content": "You are a code completion assistant."},
    {"role": "user", "content": input_text}
]
```
