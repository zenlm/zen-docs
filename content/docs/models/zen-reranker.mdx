# Zen Reranker

Zen Reranker is an 8B dual-task embedding and reranking model built on the Zen Embedding architecture. It natively outputs 7680-dimensional embeddings — the canonical dimension for the Hanzo and Zoo Networks — eliminating alignment overhead in Decentralized Semantic Optimization (DSO) pipelines. It supports 100+ languages, 8K context, and BitDelta compression.

## Specifications

| Property | Value |
|----------|-------|
| Parameters | 8.2B (8B base + 200M projection head) |
| Embedding Dimension | 7680 (native, no alignment needed) |
| Context Length | 8,192 tokens |
| Languages | 100+ |
| License | Apache 2.0 |
| Tasks | Embedding + Reranking (dual-task) |

## Model Variants

## Quick Start

```python
from zen_reranker import ZenReranker

model = ZenReranker.from_pretrained("zenlm/zen-reranker-8b")

# Generate 7680-dim embeddings
texts = [
    "How to handle async errors in Rust?",
    "Explain gradient descent",
]

embeddings = model.encode(texts)
print(embeddings.shape)  # (2, 7680)
```

## Reranking

```python
query = "async error handling in Rust"
docs = [
    "Use Result<T, E> with ? operator",
    "Tokio provides timeout utilities",
    "Python has try-except blocks",
    "Result type propagates errors"
]

scores = model.rerank(query, docs)
ranked = sorted(zip(docs, scores), key=lambda x: x[1], reverse=True)
```

## Architecture

Zen Reranker extends Zen Embedding 8B with:

1. **Expanded projection head** — 4096 → 7680 dimensions via learned linear projection
2. **Dual-task training** — Contrastive embedding loss + cross-encoder reranking loss
3. **BitDelta optimization** — 31.87x compression (964 bytes per embedding) with 97.3% quality retention
